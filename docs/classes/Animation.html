Defined in js/foundation/animation.js:33

Parameters:

  • drawable Drawable

    The drawable to animate.

  • frameFunction Function

    A function that updates the animation. Return false to terminate the animation. It may take the duration in milliseconds from the beginning of the animation as a parameter. For example, 

    function(time) {
    shape.x = Math.floor(time / 100);
    return time <= 1000;
}

  • callback Function

    A function to perform at the completion of the animation.

Class: Animation

foundation/animation. Animation

new Animation(shape, frameFunction, numFrames, callback)

Animation
Parameters:
Name Type Description
shape Shape The shape to animate.
frameFunction function A function that updates the animation. Must return a boolean, which is supposed to signal whether or not to terminate the animation (return false to terminate). It may take the frame count (starting at 0) as the first parameter and the duration in milliseconds from the beginning of the animation as the second parameter.
numFrames int The number of frames.
callback function A function to perform at the completion of the animation.
